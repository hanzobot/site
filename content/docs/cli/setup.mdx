# `bot setup`

Initialize `~/.bot/bot.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
bot setup
bot setup --workspace ~/bot
```

To run the wizard via setup:

```bash
bot setup --wizard
```
