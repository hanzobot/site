# `bot config`

Config helpers: get/set/unset values by path. Run without a subcommand to open
the configure wizard (same as `bot configure`).

## Examples

```bash
bot config get browser.executablePath
bot config set browser.executablePath "/usr/bin/google-chrome"
bot config set agents.defaults.heartbeat.every "2h"
bot config set agents.list[0].tools.exec.node "node-id-or-name"
bot config unset tools.web.search.apiKey
```

## Paths

Paths use dot or bracket notation:

```bash
bot config get agents.defaults.workspace
bot config get agents.list[0].id
```

Use the agent list index to target a specific agent:

```bash
bot config get agents.list
bot config set agents.list[1].tools.exec.node "node-id-or-name"
```

## Values

Values are parsed as JSON5 when possible; otherwise they are treated as strings.
Use `--json` to require JSON5 parsing.

```bash
bot config set agents.defaults.heartbeat.every "0m"
bot config set gateway.port 19001 --json
bot config set channels.whatsapp.groups '["*"]' --json
```

Restart the gateway after edits.
