# Features

Hanzo Bot is a full computer-using AI agent that runs on your machine or deploys to the cloud. It connects to every major chat platform and supports 100+ AI models.

## Core

- **[Runs Locally](/docs/features/local)** -- Mac, Windows, or Linux. Private by default.
- **[100+ AI Models](/docs/features/models)** -- Claude, GPT, Gemini, Llama, DeepSeek, and more.
- **[Every Chat App](/docs/channels)** -- WhatsApp, Telegram, Discord, Slack, Signal, iMessage.

## Capabilities

- **[Persistent Memory](/docs/features/memory)** -- Remembers you across sessions and becomes uniquely yours.
- **[Browser Control](/docs/features/browser)** -- Browse the web, fill forms, extract data from any site.
- **[System Access](/docs/features/system-access)** -- Read and write files, run shell commands, execute scripts.

## Extend

- **[Cloud Desktop VMs](/docs/features/cloud)** -- Full Linux desktop per agent, powered by Hanzo Cloud.
- **[MCP Integration](/docs/features/mcp)** -- 260+ tools via Model Context Protocol.
- **[Skills & Plugins](/docs/tools/skills)** -- Community skills or build your own.

## Quick start

```bash
curl -fsSL https://hanzo.bot/install.sh | bash
bot onboard
```

The onboarding wizard configures your model provider, connects a chat channel, and starts the gateway. See [Getting Started](/docs/start/getting-started) for the full walkthrough.
